# Obsidian

A premium dark gaming theme for [Azuriom](https://azuriom.com) with a visual drag-and-drop editor.

## Features

- 🎨 Visual frontend editor — edit colors, sections, and content directly on the site
- 🧱 Drag-and-drop widget system (home + vote pages)
- 🌓 Dark-first "Neon Forge" aesthetic with glow accents
- ⚙️ Configurable layouts (sidebar left/right/full width)
- 🎁 Two reward showers variants (horizontal gauge & vertical timeline)
- 🛒 Redesigned shop pages with premium game-store feel
- 📰 Enhanced article and page reading experience
- 📱 Fully responsive

## Installation

1. Copy this folder to `resources/themes/obsidian` in your Azuriom install
2. Activate the theme in **Admin > Apparence > Thèmes**
3. Click the edit button at the bottom-right to customize

## Customization

Connect as admin, then click the pencil button at the bottom-right of any page (or add `?editor=true` to the URL) to open the visual editor.
