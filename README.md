# Obsidian

A premium dark gaming theme for [Azuriom](https://azuriom.com) with a visual drag-and-drop editor.

## Features

- 🎨 Visual frontend editor — edit colors, sections, and content directly on the site
- 🧱 Drag-and-drop widget system (home + vote pages)
- ⚙️ Configurable layouts (sidebar left/right/full width)
- 🎁 Rewards Shower integration
- 🛒 Redesigned shop pages with premium game-store feel
- 📰 Enhanced article and page reading experience
- 📱 Fully responsive
- 📚 Multi-language support
